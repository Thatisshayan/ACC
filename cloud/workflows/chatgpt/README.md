# ACC / Founder Project Workflow JSON Bundle

This folder contains all workflow ideas generated during brainstorming.

Files:
- `_master_workflows.json` — all workflows in one JSON file
- individual `*.json` files — one workflow per file

Each workflow includes:
- id
- name
- category
- user command
- description
- flow steps
- parallel lanes
- final outputs
- approval rules where relevant

These can be imported into ACC, Notion, ClickUp, Lovable, Claude, Cursor, or used as starter templates for Founder Project.
